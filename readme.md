# HUST 公选课智能筛选工具

华中科技大学公选课智能筛选工具，自动检测课程时间冲突，帮你快速选择合适的公选课。

## 功能特点

- **智能冲突检测** - 自动识别与你课表冲突的公选课，一键过滤
- **内置公选课名单** - 预置 988 门公选课数据，无需手动上传
- **可视化课表** - 右侧周课表展示，直观查看现有课程与预选课程
- **预选功能** - 支持预选课程，时间冲突时自动覆盖，方便对比选择
- **收藏功能** - 收藏感兴趣的课程，方便后续查看
- **多维度筛选** - 按校区、课程类别、可选/冲突状态筛选
- **纯前端实现** - 无需后端服务器，直接打开 HTML 即可使用
- **响应式设计** - 支持电脑和手机访问

## 使用方法

1. 打开 `index.html` 文件
2. 上传你的课表文件（ICS 格式，可从 WakeUp 课程表 APP 导出）
3. 点击「开始筛选」按钮
4. 浏览筛选后的公选课列表，可按类别、校区等条件进一步筛选
5. 点击「预选」按钮可将课程添加到右侧课表中预览

## 技术实现

- **纯前端架构** - HTML + CSS + JavaScript，无需安装任何依赖
- **SheetJS (xlsx.js)** - 用于解析 Excel 文件
- **ICS/iCalendar 解析** - 解析课表文件提取课程时间信息
- **localStorage** - 本地存储收藏和预选数据
- **CSS Grid** - 实现左右分栏布局和课表网格

## 冲突检测逻辑

1. 解析 ICS 文件获取用户课程的时间信息（星期、节次）
2. 解析公选课名单获取每门课的上课时间
3. 比较用户课程与公选课的时间，检测是否存在重叠
4. 标记冲突课程，默认显示可选课程

## 注意事项

- 华科选课系统不允许选择任何时间冲突的课程，本工具据此设计
- 预选功能仅供本地预览，不会影响实际选课系统
- 收藏和预选数据保存在浏览器本地，更换浏览器需重新操作

## 开发说明

如需更新公选课数据：
1. 将新的 Excel 文件放到项目目录
2. 使用脚本将其转换为 JSON 格式
3. 将 JSON 数据内嵌到 `index.html` 中
